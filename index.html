Abstract

We present the Unified Motion-Action (UMA) Model, which uses 3D object motion trajectories as a shared interface for visuomotor control and dynamics modeling. UMA models object motion and robot actions jointly under a masked generative objective. The mask pattern determines both the supervision regime during pretraining and the inference mode at deployment. Using relabeled motion contexts and a contrastive objective that separates task intent from scene geometry, UMA supports pretraining across diverse data sources without manually annotated task instructions. At deployment, the same pretrained parameters support visuomotor control conditioned on motion, dynamics modeling from candidate actions, and task adaptation from a few demonstrations. Pretrained on robot demonstrations, human videos, and simulated data, UMA consistently outperforms leading baselines for each inference mode.

Method

UMA represents object motion as 3D points sampled on object surfaces and tracked across time, a representation that can be extracted from video without action labels, lives in the same spatial coordinate system as robot controls, and remains comparable across embodiments. A masked diffusion transformer jointly denoises motion and action tokens with a flow-matching objective, conditioned on the observation and a task latent.

One objective, heterogeneous data

Both motion and actions can serve as conditioning or as target depending on the mask pattern. Robot demonstrations supervise both streams; action-free human videos and cross-embodiment data still supervise the motion stream, so no source is wasted.

Disentangled task latent

Relabeled reference motions and a SimCLR contrastive objective produce a task latent that is invariant to keypoint sampling, temporal spacing, and SE(3) scene placement, capturing task intent rather than scene geometry.

Versatile inference

With weights fixed, changing the mask switches between visuomotor control conditioned on motion and dynamics prediction conditioned on actions. Optimizing only the task latent enables adaptation from a few demonstrations through soft prompt tuning, even from human videos without actions.

Zero-Shot Visuomotor Control

Pretrained on DROID demonstrations, human videos (HOI4D, Xperience), and simulated data, none of which overlaps the test environments, UMA outperforms the strongest specialized baseline by 20–25 percentage points on every task, spanning 6-DoF manipulation, tool use, and deformable objects. Predicting Dynamics from Motion

Conditioned on candidate actions, the same pretrained model predicts the resulting object motion. This supports model predictive control and exposes whether predicted dynamics match the task intent before any action is executed. UMA reduces average prediction error from 0.054 (Point-World) to 0.042, and removing simulated data from pretraining increases error by nearly 5×.

Adapting from a Few Demonstrations

UMA adapts to new tasks by optimizing only the task latent on 25 demonstrations while keeping all model weights frozen. With action labels, it matches or beats π0.5-LoRA and finetuned UVA. With human videos alone and no action labels, it outperforms UVA by ~25 percentage points on every task, without any robot data for the new task.

Method in Detail

Extracting 3D Motion Supervision from Any Video

UMA's pretraining data comes from monocular RGB videos, including robot demonstrations (DROID), human videos (HOI4D, Xperience), and cross-embodiment sources, with no ground-truth depth, camera intrinsics, or action labels required. A four-stage pipeline converts raw video into 3D keypoint trajectories. First, MegaSaM jointly estimates per-frame camera poses, intrinsics, and up-to-scale depth. Second, UniDepth anchors the depth sequence to metric scale via a global scale-and-shift fit. Third, a vision-language model identifies the task-relevant object and SAM 3 segments it, after which keypoints are sampled densely on the object surface plus sparsely in the background for scene context. Fourth, TAPIP3D tracks the sampled points to produce persistent 3D keypoint trajectories that serve as motion supervision.

Task Specification Beyond Reference Motion

Because the contrastive objective trains the task latent to capture intent independently of the modality that specifies it, alternative encoders can plug into the same latent space without retraining the policy. For language-conditioned control (left), a SigLIP2 text encoder and a lightweight aggregation transformer map an instruction directly to task tokens that replace the motion-derived latent. For goal-conditioned control (right), SAM 3 segments the described object in the initial and goal images, RoMa v2 matches points between them, and the matches are lifted to 3D, yielding a sparse two-timestep reference motion that feeds the standard motion-conditioned pathway.

Masked DiT Blocks

A single transformer pass must serve two conflicting roles. Target tokens (masked motion or actions) need to be denoised from noise toward the data manifold, while given tokens (clean conditioning) should pass through with minimal corruption. A standard DiT block applies one set of adaLN modulation weights to all tokens, forcing a compromise. Masked DiT blocks instead maintain two independent modulation parameter sets conditioned on the same diffusion timestep, applying a denoising-appropriate scale and shift to targets while preserving clean conditioning signals. Structured spatial, temporal, and context attention replaces dense global attention.
